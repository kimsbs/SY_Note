HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the buulding blocks of HTML pages. With HTML constructs, images and orhter objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structed documents by denoting structural semantics for text such as heading, paragrahs, lists, lins, quotes and other items. HTML elements are delineated by tags. written using angle brackets.
